Spring Boot Actuator는 Spring 모듈 중 하나입니다.

상태 점검, 메트릭 수집, HTTP 추적 등과 같은 프로덕션에 사용 가능한 기능을 제공하여,

Spring Boot 애플리케이션을 모니터링하고 관리하는 데 도움을 줍니다.

JMX 또는 HTTP 엔드포인트를 통해 액세스할 수 있고,

외부 애플리케이션 모니터링 시스템과 통합할 수도 있습니다.
